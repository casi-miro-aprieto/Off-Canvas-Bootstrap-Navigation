Sliding-Menu-CSS
================

Simple SASS menu that slides from the side to show the menu options.

![alt tag](https://github.com/jycr753/Sliding-Menu-CSS/blob/master/image.jpg)

Browser  | Support
------------- | -------------
|Chrome 38  | True |
|Safari 7  | Sort of |
|Firefox  | True |
|IE < 10  | False |
